# House Boston Price
- Numpy.ipynb : Xây dựng mô hình dự đoán nhà bằng BGD, SGB, mini-BGD bằng thư viện numpy.
- Numpy.ipynb : Xây dựng mô hình dự đoán nhà bằng BGD, SGB, mini-BGD bằng framework tensorflow.
- Preprocess_Data.ipynb : Tiền xử lý dữ liệu sang dạng dữ liệu ma trận.
- housing.data : Data thô các số liệu về giá nhà ở Boston dựa trên 13 thuộc tính.
- housing.names : Giới thiệu về data.
+ Các file dạng nhị phân:
- train_X.npy : Tập dữ liệu các thuộc tính để train với kích thước 336x13
- test_X.npy : Tập dữ liệu các thuộc tính để test với kích thước 170x13
- train_y.npy : Tập dữ liệu các nhãn để train với kích thước 336x1
- test_y.npy : Tập dữ liệu các nhãn để test với kích thước 170x1
+ Các file dạng thập phân:
- train_X.txt : Tập dữ liệu các thuộc tính để train với kích thước 336x13
- test_X.txt : Tập dữ liệu các thuộc tính để test với kích thước 170x13
- train_y.txt : Tập dữ liệu các nhãn để train với kích thước 336x1
- test_y.txt : Tập dữ liệu các nhãn để test với kích thước 170x1
